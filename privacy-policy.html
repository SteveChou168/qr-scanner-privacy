隱私權政策

最後更新日期：2025 年 1 月 30 日

摘要：QR Scanner 是一款離線優先的應用程式。您的掃描資料儲存在您的裝置上，不會上傳至任何伺服器。我們重視您的隱私。

1. 資料收集

1.1 我們不收集的資料

  • 我們不會收集您的個人身份資訊
  • 我們不會上傳您的掃描記錄至雲端
  • 我們不會追蹤您的精確位置座標
  • 我們不會存取您的聯絡人或其他個人資料

1.2 本機儲存的資料

以下資料僅儲存在您的裝置上：

  • 掃描記錄：QR Code / 條碼的內容、掃描時間
  • 位置資訊（選用）：僅儲存城市/區域名稱，不儲存精確座標
  • 圖片（選用）：您選擇儲存的 QR Code 圖片
  • 應用程式設定：主題、語言等偏好設定

2. 權限使用說明

2.1 相機權限

用於掃描 QR Code 和條碼。相機畫面僅在本機處理，不會傳輸至外部伺服器。

2.2 位置權限（選用）

用於記錄掃描地點的城市/區域名稱。您可以隨時在系統設定中關閉此權限。我們僅儲存地名（如「台北市大安區」），不儲存 GPS 座標。

2.3 儲存權限

用於將產生的 QR Code 圖片儲存至您的相簿。

2.4 網路權限

用於顯示廣告及開啟掃描結果中的網址連結。

3. 廣告

本應用程式使用 Google AdMob 顯示廣告。Google 可能會收集以下資訊用於廣告投放：

  • 裝置識別碼
  • IP 位址
  • 廣告互動資料

詳情請參閱 Google 隱私權政策。

4. 資料安全

  • 所有掃描資料都儲存在您裝置的本機資料庫中
  • 我們不營運任何雲端伺服器來儲存使用者資料
  • 解除安裝應用程式將會刪除所有本機資料

5. 兒童隱私

本應用程式不針對 13 歲以下兒童。我們不會故意收集兒童的個人資訊。

6. 隱私權政策變更

如有任何變更，我們會在此頁面更新。建議您定期查看此政策。

7. 您的權利

  • 存取權：您可以在應用程式內查看所有儲存的資料
  • 刪除權：您可以在應用程式內刪除任何掃描記錄
  • 選擇退出：您可以隨時關閉位置權限
