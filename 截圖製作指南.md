## Feature Graphic 設計（Google Play）

**尺寸**：1024 x 500 px

**建議設計元素**：
- App 名稱「QR Scanner」或「QR 掃描器」
- App 圖示
- 核心功能關鍵字：「Offline」「AR Mode」「Privacy First」
- 簡潔背景（漸層或純色）
- 掃描示意圖（可選）

**範例佈局**：
```
+-------------------------------------------+
|                                           |
|  [App Icon]  QR Scanner                   |
|                                           |
|  Offline • AR Mode • Privacy First        |
|                                           |
+-------------------------------------------+
```

## 截圖設計規範

### 文字規範

| 項目 | 建議 |
|------|------|
| 標題字體大小 | 48-72px |
| 副標題字體大小 | 24-36px |
| 字體 | 系統字體或無襯線字體 |
| 文字位置 | 上方或下方，不遮擋主要內容 |

### 色彩建議

| 元素 | 建議 |
|------|------|
| 背景 | 與 App 主題色一致或漸層 |
| 文字 | 白色或深色（需有對比） |
| 強調色 | 使用 App 的品牌色 |

### 設計技巧

1. **裝置框架**：將截圖放入手機模型中更專業
2. **一致性**：所有截圖使用相同的設計風格
3. **對比度**：確保文字在背景上清晰可讀
4. **留白**：不要塞太多元素，保持簡潔
5. **行動呼籲**：第一張截圖最重要，展示核心功能

## 各語言截圖製作清單

### 必須製作（7 語言 × 8 張 = 56 張）

| 語言 | 代碼 | 狀態 |
|------|------|------|
| 繁體中文 | zh-Hant | ⬜ |
| English | en | ⬜ |
| 日本語 | ja | ⬜ |
| Español | es | ⬜ |
| Português | pt | ⬜ |
| 한국어 | ko | ⬜ |
| Tiếng Việt | vi | ⬜ |

### 製作流程

1. **設定 App 語言**：在 Settings 中切換到目標語言
2. **截取原始畫面**：使用模擬器或實機截圖
3. **加入設計元素**：使用 Figma/Canva 加入文案和框架
4. **匯出**：確保尺寸正確，格式為 PNG 或 JPEG
5. **命名**：使用統一命名規則，如 `01_scan_zh.png`

---

## 截圖命名規則

建議格式：`{序號}_{畫面}_{語言}.png`

範例：
```
01_scan_zh.png
01_scan_en.png
01_scan_ja.png
02_ar_mode_zh.png
02_ar_mode_en.png
...
```

## 注意事項

1. **不要使用假資料**：截圖中的內容應該是真實可見的功能

2. **避免敏感資訊**：確保截圖中沒有個人資料或真實位置

3. **展示真實功能**：不要展示 App 沒有的功能

4. **測試不同裝置**：確保截圖在不同尺寸上都好看

5. **定期更新**：當 App 有重大更新時，記得更新截圖
